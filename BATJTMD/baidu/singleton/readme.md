# 单例模式
-类和实例
    -JS本身没有类
    class + new 不是必须的，JS有对象字面量和原型链
    function+prototype
    es6 js 用起来像java
    -类是抽象，对象是具体的
        一个类的两个实例，不是同一对象
-百度考题
    不管new多少次，它都只给你返回第一次所创建的那唯一一个实例
    -一般的面向对象，设计模式  23种常见模式，大型项目开发中常年积累下来
        解决中大型面向对设计的范式
如何使用单例模式？
    -static 方法，绕开了直接new
    -返回值就是第一次且唯一一次new的实例，挂载在类上
